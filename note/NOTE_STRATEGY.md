## フォロワー増加戦略

### 投稿設計
- **頻度**: 週3〜5本（最初の3ヶ月は毎日投稿が理想）
- **構成**: 短め記事（800〜1500字）を高頻度 ＋ 月2本の渾身ロング記事（3000〜5000字）
- **テーマ**: AI副業・Claude活用・自動化・副収入に一貫して絞る

### タイトル法則（note編集部500件分析より）
- **文字数**: 15〜25文字
- **構造**: 数字 ＋ 固有名詞（AI/Claude/n8n）＋ ターゲット ＋ ベネフィット
- **キーワード**: タイトル文頭に配置

**勝ちパターン例**
```
「月5万稼ぐAI副業、私が3ヶ月で辿り着いた最短ルート」
「ChatGPTで自動化できた7つの作業【副業初心者向け】」
「スキルゼロからClaude Codeで月3万達成した全手順」
「n8n×AIで1日15分の副業自動化を設計した話」
```

### プロフィール設計（必須）
- 冒頭に実績・数字・専門性を明記
  - 例:「AI自動化で副業月○万円 / n8n・Claude Code歴○年 / 初心者でも再現できる手順を毎週公開」
- ターゲット読者を一行で明示

## コンテンツ設計

### 無料 → 有料の導線設計

```
無料記事（週3〜5本）
  → ノウハウの「型・考え方・一部手順」を公開
  → 末尾CTA:「詳細な手順・テンプレ・設定ファイルは有料マガジンにまとめました」

有料記事・マガジン
  → 実際に動くコード・設定ファイル・詳細プロセス・テンプレートを販売
  → 単品 1500〜2000円 or マガジン 3980円

メンバーシップ（月額）
  → 毎週の実況レポート・限定コンテンツ・質問対応
  → 500〜1000円/月
```

### 有料ラインの設置場所
- 記事の「最もおいしい部分の手前」に有料壁を設置
- 読者の期待が最高潮に達したタイミングで課金を促す

### 1次情報の必須化
- 毎記事に「自分が試した結果の数値」を最低1つ入れる
- AI生成のみの記事は購買率が低く、編集部おすすめにも選ばれにくい
- 例:「実際にClaude Codeを使って○時間が○分に短縮」「先月の副業収益○万円の内訳」

## X → note 誘導戦略

### Xリンク掲載のルール（重要）
- **本文にURLを入れるとリーチが激減**（X アルゴリズム対策）
- リンクは**リプライ欄**に貼る、またはプロフィールのURLに固定

### 効果的なXポストパターン（3種類）

**1. ティーザー型（最多使用）**
```
[ポスト本文]
ChatGPTで自動化できた7つの作業を全公開します

① ブログ記事の下書き
② メール返信の文案
③ データ整理
...

（続きはnoteにまとめました）
↓
[リプライ欄にnoteリンク]
```

**2. 価値先出し型**
```
[ポスト本文]
n8nでXを自動化する最小構成：

- トリガー: Cron（毎朝7時）
- Claude API でツイート生成
- xurl で投稿

詳細な設定ファイル・コード全文はnoteの有料記事にまとめました
↓
[リプライ欄にnoteリンク]
```

**3. 実績報告型**
```
先月noteだけで○万円達成しました

使ったのはこの3つだけ：
・毎週のリアル収益レポート（有料）
・AIで作った設定テンプレ販売
・月額メンバーシップ

詳細 → プロフリンクのnoteへ
```

## noteアルゴリズム攻略（2026年版）

### 検索・おすすめに効く要素
1. **タイトル文頭のキーワード**（「AI副業」「Claude活用」等）
2. **公開後24〜48時間の初速**（スキ・コメント数が評価軸）
3. **継続投稿頻度**（アクティブ作者が優遇）
4. **ハッシュタグ**: 1記事3〜5個
5. **記事長**: 3000字以上でGoogle SEO流入が発生（noteはGoogleドメイン評価が高い）

### 編集部おすすめ枠のルート
- 1次情報（体験談・数字）必須
- タイトル・見出し画像・冒頭の質で手動キュレーション
- シリーズ化 → マガジン化でウォッチリストに入りやすい
- 公式コンテスト（創作大賞等）への参加

### SEOターゲットキーワード（優先度順）
```
「AI副業 始め方 初心者」
「Claude Code 副業」
「n8n 自動化 副収入」
「ChatGPT 副業 月5万」
「生成AI 副業 2026」
「プログラミングなし AI副業」
```
