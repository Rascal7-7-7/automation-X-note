## 12. Google Cloud Console 設定（API連携用）

YouTube Data API v3 の OAuth2 設定時に必要な情報：

| 項目 | 値 |
|---|---|
| アプリ名 | ぬちょ AI副業ハック（内部用途） |
| スコープ | youtube.upload, youtube.readonly |
| リダイレクトURI | http://localhost:3001/auth/youtube/callback |

> `.env` に追記するキー:
> ```
> YOUTUBE_CLIENT_ID=
> YOUTUBE_CLIENT_SECRET=
> YOUTUBE_REFRESH_TOKEN=
> YOUTUBE_CHANNEL_ID=
> ```
