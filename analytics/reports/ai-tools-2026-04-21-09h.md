## 注目ツール TOP3

### ToolHive Studio（出典: GitHub stacklok/toolhive-studio ★121）
- **概要**: MCPサーバーのインストール・管理・実行をGUIで一元化するデスクトップアプリ。AIエージェントへの接続も管理可能。
- **適用場面**: n8n + Bridge Server + Playwright MCP など複数MCPが乱立するこのプロジェクトで、各サーバーの起動状態・設定・バージョンをPM2と並行して可視化管理。Claude Code CLIとの接続設定をGUIで確認できるため、デバッグ工数を削減。
- **導入難易度**: 低
- **推奨**: 今すぐ試す

---

### Scraper MCP（出典: GitHub jessaminesimple608/scraper-mcp ★3）
- **概要**: Webスクレイピング結果をAI向けにフィルタリング・トークン最適化して返すMCPサーバー。
- **適用場面**: A8.netアフィリエイト同期で現在Playwrightフル起動しているクローリング処理の一部を軽量化。商品情報・報酬単価の定期取得など「ページ全体は不要で特定データだけ欲しい」ケースでHaikuへ渡すトークンを削減し、コスト圧縮に直結。
- **導入難易度**: 低
- **推奨**: 様子見（★3で実績薄いため、Playwright MCPとの役割分担を検証してから判断）

---

### Claude Code Manager（出典: GitHub vishalguptax/claude-code-manager ★3）
- **概要**: VS Code/Cursor/WindsurfサイドバーからClaude Codeセッション・MCPサーバー・フック・エージェントを管理するVS Code拡張。
- **適用場面**: Claude Code CLIで多視点ペルソナ5並列レビューを回す際のセッション管理を視覚化。コマンド・フック設定をGUIで管理することでn8nワークフローからのCLI呼び出しパラメータのミス削減に貢献。
- **導入難易度**: 低
- **推奨**: 様子見（★3・新規リポジトリのため安定性要確認）

## 業界トレンド

MCP周辺は「乱立・断片化」フェーズに入っており（Reddit: "MCP servers too fragmented"）、個別ツールの品質よりも**管理・統合レイヤー**（ToolHive、mcp-linkerなど）への需要が急上昇中。またAnthropicリレー経由の10秒遅延報告が散見されており、**ローカルMCP優先 + Bridge Server直結**のアーキテクチャ判断が今後の安定稼働のカギになる。
