## 🎯 成功のポイント

### ✅ やるべきこと

1. **数値を入れる** → 「月5万」「3ヶ月で」など具体的
2. **プロセスを見せる** → 失敗→成功のストーリー
3. **初心者に優しく** → 「初心者向け」と明示し、期待値を正しく
4. **note へ誘導する** → 毎週のCTA、プロフィール、リンク
5. **正直に答える** → リプライへの返信は丁寧に、質問に答える

### ❌ 避けるべきこと

1. **過度な保証** → 「絶対稼げる」は言わない
2. **AI感の出しすぎ** → 「人間として発信」が方針
3. **バズを狙いすぎ** → 無理なジャンル変更をしない
4. **投稿の質より量** → 毎日投稿よりも週5の質投稿
5. **フォロワー数の見栄張り** → 正直な数字で信頼構築

## ⚠️ トラブルシューティング

### X投稿が反応しない場合

**原因**: テンプレートが「個人の実績」に合っていない

**解決**:
1. 自分の実績を正確に確認
2. テンプレートの数字を正しく埋める
3. 1週間様子を見る（運用初期は反応が少ないのが正常）
4. バズツイート（いいね100+）の型を分析して微調整

### note が売れない場合

**原因**: 記事タイトル or CTA が弱い可能性

**解決**:
1. note タイトルに「月5万」「失敗」など SEO キーワード
2. X での告知ツイートをより具体的に（「誰向け」「何が得られるか」を明示）
3. 記事の冒頭に「この記事で得られる3つのこと」をリスト
4. 価格を1000円 → 500円に下げて、まずは「購買実績」を作る

### Instagram リール が伸びない場合

**原因**: 完了率が低い可能性（テンポが遅い、内容が退屈）

**解決**:
1. 動画を30秒以下に短縮
2. 最初の3秒で「これは何の動画か」を明示
3. テンポよく、シーン転換を多く
4. ナレーション or テキスト解説で「何を学べるか」を説明
